# User CRUD API

A simple RESTful API for managing users with CRUD operations, built using Node.js, Express, and MongoDB.

## Features
- Create, Read, Update, and Delete (CRUD) users.
- Data validation using Zod.
- MongoDB integration with Mongoose.
